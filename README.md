# TDS_ROE_6

This repository contains the GitHub Actions workflow for **Multi-Platform Matrix Build with Artifacts**.

It demonstrates:
- Parallel matrix builds for Node.js versions 14, 16, and 18  
- Artifact uploads per build variant (build-f736404-<version>)
- Automated CI/CD workflow using GitHub Actions
